==== From Remote ====


Execute MSWE agent:
```
python run.py \
   --model_name gpt4o \
   --cache_task_images True \
   --pre_build_all_images True \
   --remove_image False \
   --pr_file /Users/vartiukhov/dev/studies/hse/thesis/thesis-metamorphic-eval/benchmarks/metamorphic-mockito.jsonl \
   --config_file /Users/vartiukhov/dev/studies/hse/thesis/thesis-metamorphic-eval/agents/MSWE-agent/config/default.yaml \
   --skip_existing=True \
   --per_instance_cost_limit 5.00 \
   --print_config=False \
   --max_workers_build_image 16
```

Execute MSWE agent (with local repos):
```
python run.py \

   --cache_task_images True \
   --pre_build_all_images True \
   --remove_image True \

   --config_file config/default.yaml --skip_existing=True \
```
